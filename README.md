# Home Finder Prototype

Prompt-first property finder MVP built in React + Vite.

## What it does
- Lets users type natural-language property prompts
- Parses bedrooms, budget, location, property type, and requested features
- Returns only relevant listings
- Shows why a property matches
- Includes a lightweight booking flow

## Local run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to GitHub + Vercel
1. Create a new GitHub repository
2. Upload these files to the repo root
3. In Vercel, click **Add New Project**
4. Import the GitHub repository
5. Vercel should detect **Vite** automatically
6. Click **Deploy**

## Demo prompts
- 1 bed flat in Hyde Park under £600k
- 1 bed flat with gym near tube
- 2 bedroom property with a fireplace under £600k
- 1 bed apartment in Westminster with gym near metro under £500k
